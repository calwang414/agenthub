# AI 智能体插件市场 · 后台管理 — 插件管理页面

## 页面目标
为 AI 智能体插件市场提供后台管理界面，管理员可以在此页面查看、搜索、筛选、新增、编辑、删除插件。

## 整体布局结构
- 左侧垂直导航栏（placeholder 占位，本次不做交互）：仪表盘 / 插件管理（当前激活） / 分类管理 / 用户管理
- 右侧主内容区：顶部操作栏 + 视图切换 + 插件列表

## 核心功能

### 1. 侧边导航
- 深色底侧边栏，Logo 在顶部
- 4 个导航项：仪表盘、插件管理（高亮当前）、分类管理、用户管理
- 底部显示管理员头像和名称
- 可折叠（移动端适配）

### 2. 顶部操作栏
- 页面标题 "插件管理"
- 搜索框（支持按名称或作者搜索插件）
- 新增插件按钮（coral 主色按钮）
- 分类筛选下拉（全部 / Skill / Agent / Tool / MCP 等）
- 状态筛选：全部 / 已上架 / 已下架 / 待审核

### 3. 视图切换
- 表格视图 / 卡片视图 切换按钮
- 表格视图：列表形式展示插件，包含名称、版本、作者、分类、下载量、状态、操作
- 卡片视图：网格卡片展示插件，卡片显示封面占位图、名称、作者、下载量、状态标签、操作按钮

### 4. 插件表格（表格视图）
- 列：名称（带图标）、版本号、作者、分类标签、下载量、评分、状态、操作
- 操作列：编辑、上下架开关、删除（确认弹窗）
- 支持多选批量操作（批量上架/下架/删除）
- 分页控件

### 5. 插件卡片（卡片视图）
- 网格布局，每行 3-4 个卡片
- 卡片内容：插件图标/封面占位、名称、简短描述、作者、下载量、评分星级、状态标签
- hover 效果：轻微上浮 + 阴影
- 卡片内嵌：编辑 / 上下架 / 删除 快捷操作

### 6. 新增/编辑插件弹窗
- 弹窗表单：插件名称、描述、作者、分类（下拉）、版本号、标签（多选/输入）
- 封面图上传占位区
- 保存 / 取消按钮
- 表单校验（名称必填）

### 7. 删除确认
- 确认弹窗："确定要删除插件「XXX」吗？此操作不可撤销。"
- 确认 / 取消按钮

### 8. 状态切换
- 上架/下架开关（Switch/Toggle 组件）
- 实时切换并更新状态标签

### 9. 数据统计卡片
- 顶部 4 个统计卡片横排：插件总数、已上架数量、今日下载量、待审核数量
- 每个卡片带图标和简要数字

## 交互行为
- 搜索框输入实时筛选（防抖 300ms）
- 分类和状态下拉切换即时刷新列表
- 表格/卡片视图切换记忆状态
- 新增/编辑弹窗支持 ESC 关闭
- 删除操作需二次确认
- 批量操作选中后顶部浮现批量操作栏
- 所有操作有成功/失败的 toast 反馈
